# AI-Powered Sudoku Solver 🧩
A Sudoku solver that uses Gemini-2.5-Flash for OCR and a SAT Solver for logical solving.

This project is a high-performance Sudoku solver that combines Artificial Intelligence for image recognition and discrete mathematics for logical solving.

## 🌟 Features
* **AI-Based OCR:** Uses **Gemini-2.0-Flash** to recognize digits from uploaded Sudoku images.
* **Image Processing:** Pre-processes images using **PIL (Pillow)** for better accuracy (grayscale, contrast, and sharpening).
* **Logical Engine:** Implements a **SAT Solver** (Discrete Mathematics approach) to solve the 9x9 grid instantly.
* **User Interface:** Simple and functional GUI built with **Tkinter**.

## 🛠️ Built With
* Python 3.x
* Google Generative AI API (Gemini)
* Pillow (PIL)
* Tkinter

## 🚀 How to Run
1. Clone the repository.
2. Install requirements: `pip install google-generativeai Pillow`.
3. Add your Gemini API Key to `Main.py`.
4. Run `python Main.py`.
